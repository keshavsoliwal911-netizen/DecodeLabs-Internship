# Python To-Do List

A simple command-line To-Do List application built with Python. This project demonstrates the use of lists, loops, conditional statements, and user input handling.

## Features

* Add new tasks
* View all tasks
* Store multiple tasks using a Python list
* Menu-driven interface
* Beginner-friendly code

## Project Structure

```text
Project_1/
├── main.py
└── README.md
```

## Code Overview

The program uses:

* **Lists** to store tasks
* **append()** to add new tasks
* **while loop** to keep the program running
* **for loop** to display all tasks
* **if-elif-else** statements for menu navigation

## How to Run

1. Make sure Python is installed on your system.
2. Clone the repository:

```bash
git clone https://github.com/keshavsoliwal911-netizen/DecodeLabs-Internship.git
```

3. Navigate to the project folder:

```bash
cd Project_1
```

4. Run the program:

```bash
python main.py
```
