# Random Password Generator

A simple Python project that generates secure random passwords based on the length given by the user.

## Features

* Generates random passwords of any user-defined length
* Uses uppercase letters (A-Z)
* Uses lowercase letters (a-z)
* Uses numbers (0-9)
* Validates user input
* Generates a different password every time the program runs

## Technologies Used

* Python 3
* `secrets` module
* `string` module

## How It Works

1. The user enters the desired password length.
2. The program reads the input.
3. A character pool containing letters and digits is created.
4. A secure random password is generated.
5. The generated password is displayed on the screen.
